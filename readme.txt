== Description ==

Tricks & Forms is a custom posts, profiles and options screens building tool. It doesn't have a user interface, but a programming interface instead, id est, a PHP variables and functions set.

== Installation ==

1. Download tricks-forms.zip;
1. Upload tricks-forms.zip (Dashboard -> Plugins -> Add New -> Upload).
